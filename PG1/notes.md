# C Notizen

## Formatierungszeichen

| Zeichen | Beschreibung            |
|---------|-------------------------|
| %c      | Zeichen                 |
| %d      | Dezimalzahl (int)       |
| %e      | Wissenschaftliche Notation (float) |
| %f      | Dezimalzahl (float)     |
| %g      | Kürzeste Darstellung (float) |
| %i      | Dezimalzahl (int)       |
| %o      | Oktalzahl               |
| %p      | Zeigeradresse           |
| %s      | Zeichenkette            |
| %u      | Unsigned Dezimalzahl    |
| %x      | Hexadezimalzahl (klein) |
| %X      | Hexadezimalzahl (groß)  |
